# W3nbo Xu Portfolio

A minimalist, high-performance personal brand portfolio for a product manager.

## Tech Stack

- Next.js 14 + React 18 + TypeScript
- Tailwind CSS v3
- Lenis (smooth scroll)
- GSAP + ScrollTrigger (scroll-driven animations)
- Framer Motion (component micro-interactions)
- Lucide React (icons)

## Features

- Apple-style dark minimalism with blue accent
- Lenis smooth scrolling with GSAP ScrollTrigger sync
- Hero particle background with mouse gravity
- Character-by-character hero title reveal + typewriter identity switcher
- 3D tilt work cards with dynamic cursor lighting
- Scroll-triggered section reveals (fade-up, blur-in, slide)
- Hamburger-to-close menu with clip-path full-screen overlay
- Ripple-effect magnetic buttons
- Spring-physics back-to-top button
- Skill constellation visualization
- Reduced-motion support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

npm run build
